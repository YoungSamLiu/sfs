# 修改文件系统绑定的安全组<a name="ChangeSecurityGroup"></a>

## 功能介绍

修改SFS Turbo文件系统绑定的安全组。修改安全组为异步任务，可以通过“查询单个文件系统”返回的子状态字段“sub\_status”来判断是否修改安全组状态，子状态为“232”即为修改安全组成功。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SFSTurbo&api=ChangeSecurityGroup)中调试该接口。

## URI

POST /v1/\{project\_id\}/sfs-turbo/shares/\{share\_id\}/action
