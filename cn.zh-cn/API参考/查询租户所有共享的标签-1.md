# 查询租户所有共享的标签<a name="ListSharedTags"></a>

## 功能介绍

查询租户所有共享的标签集合。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SFSTurbo&api=ListSharedTags)中调试该接口。

## URI

GET /v1/\{project\_id\}/sfs-turbo/tags
