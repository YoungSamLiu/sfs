# 查询文件系统详细信息<a name="ShowShare"></a>

## 功能介绍

查询SFS Turbo文件系统详细信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SFSTurbo&api=ShowShare)中调试该接口。

## URI

GET /v1/\{project\_id\}/sfs-turbo/shares/\{share\_id\}
