# 查询共享标签<a name="ShowSharedTags"></a>

## 功能介绍

查询指定共享的所有标签信息。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SFSTurbo&api=ShowSharedTags)中调试该接口。

## URI

GET /v1/\{project\_id\}/sfs-turbo/\{share\_id\}/tags
