# 创建共享<a name="ZH-CN_TOPIC_0076922188"></a>

## 功能介绍<a name="zh-cn_topic_0064390791_section3086974817377"></a>

创建文件共享。

>![](public_sys-resources/icon-note.gif) **说明：**   
>该接口为异步接口，返回为200只是表示接口下发接收成功，后续可以通过[查询共享详细信息](查询共享详细信息.md)查询共享的共享状态和共享路径，判断创建共享是否完成以及是否成功。  

>![](public_sys-resources/icon-notice.gif) **注意：**   
>通过该接口创建共享成功后，用户还需要参考[添加共享访问规则](添加共享访问规则.md)添加共享访问规则，之后才可正常使用创建的共享。
