# 删除共享标签<a name="DeleteSharedTag"></a>

## 功能介绍

指定共享删除一个标签。当共享中不存在指定要删除的key时，接口调用将会返回404错误。

## 调试

您可以在[API Explorer](https://apiexplorer.developer.huaweicloud.com/apiexplorer/doc?product=SFSTurbo&api=DeleteSharedTag)中调试该接口。

## URI

DELETE /v1/\{project\_id\}/sfs-turbo/\{share\_id\}/tags/\{key\}
