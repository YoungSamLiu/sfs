# 批量添加共享标签<a name="ZH-CN_TOPIC_0200310191"></a>

## 功能介绍<a name="section65483112161"></a>

指定共享批量添加标签。

一个共享上最多有10个标签。

一个共享上的多个标签的key不允许重复。

此接口为幂等接口：如果要添加的key在共享上已存在，则覆盖更新标签。
